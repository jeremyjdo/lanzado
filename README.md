# Lanzado

Application web de voyage pour planifier un séjour en famille à Lanzarote (Îles Canaries).

## Fonctionnalités

- **Carte interactive** - Visualisez tous les points d'intérêt avec Leaflet.js
- **To Do List** - Itinéraire organisé par catégories (incontournables, famille, plages, spots photo)
- **Guide Food** - Spécialités canariennes et recommandations culinaires
- **Traduction** - Phrases essentielles espagnol-français

## Tech Stack

- HTML5 / CSS3 / JavaScript vanilla
- Leaflet.js pour la cartographie
- Lucide Icons
- LocalStorage pour la persistance

## Déploiement

L'application est un fichier HTML unique, déployable sur n'importe quel hébergeur statique.

```bash
# Avec Vercel
vercel

# Ou simplement ouvrir index.html dans un navigateur
```
